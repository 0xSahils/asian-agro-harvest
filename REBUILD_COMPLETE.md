## 📝 How to Customize Further

### Change WhatsApp Number
Edit `/components/WhatsAppButton.tsx`:
```tsx
const phoneNumber = '919876543210'; // Your number here
```

### Add More Animations
Add these classes to elements:
```tsx
className="fade-in-up hover-lift"
style={{ animationDelay: '0.2s' }}
```

### Use Images on Other Pages
```tsx
import Image from 'next/image';

<Image
  src="/images/global-shipping.jpg"
  alt="Description"
  fill
  className="object-cover"
/>
```

### Change Fonts
- Headings: Use `font-playfair` class
- Body: Use `font-dm-sans` class
- Mix: Combine both for specific styling
