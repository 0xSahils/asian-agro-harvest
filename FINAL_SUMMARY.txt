 HOW TO USE
================================================================================

1. RUN LOCALLY:
   npm install
   npm run dev
   → Visit http://localhost:3000

2. VIEW HOMEPAGE:
   • All luxury design implemented
   • All animations working
   • All AI images integrated
   • WhatsApp button visible (bottom-right)

3. TEST RESPONSIVE:
   • Resize browser to see mobile menu
   • Check tablet layout
   • Full desktop features

4. CUSTOMIZE WHATSAPP:
   Edit /components/WhatsAppButton.tsx:
   const phoneNumber = 'YOUR_WHATSAPP_NUMBER'; // Line 5

5. ADD WHATSAPP TO ALL PAGES:
   Import and add to any page:
   <WhatsAppButton />

6. USE COLOR SYSTEM:
   All Tailwind classes available:
   bg-primary, bg-accent, text-primary, etc.

7. USE FONTS:
   Headings:  className="font-playfair text-5xl font-bold"
   Body:      className="font-dm-sans text-lg"

8. USE ANIMATIONS:
   <div className="fade-in-up">Content</div>
   <div className="fade-in-down">Content</div>
   <button className="hover-lift">Click</button>
